# Snake Cube Analysis

Written with [vitepress](https://vitepress.vuejs.org/).

## User Guide

Commands:

```sh
# intall
pnpm install

# dev
pnpm run doc:dev

# build
pnpm run doc:build

# deploy
chmod +x deploy-ghp.sh
# warning: run after commit
./deploy-ghp.sh
```

NOTE: Use [Volar Takeover Mode](https://vuejs.org/guide/typescript/overview.html#volar-takeover-mode).
